Purpose

The purpose of this site is to showcase my application development capability using Python 3 and modern software development practices. Too often, we see nice looking web sites/applications that have code readability issues plus more. This can be due to many reasons, most of which are resolvable by adopting modern software development practices. In this case, I use these skills to develop a web app in Python 3.6.4 that uses geopy to get longitude and latitude and run inverse searches for addresses using geo coordinates as an initial feature offering.

Future Application Versions

I have received some great feedback on the beta version of this web application and have put together a list of future enhancements and features that would extend this application in a meaningful way.
